顺序表

顺序表是线性表的顺序存储结构，在内存中用地址连续的一块存储空间顺序存放线性表的各个元素。内存中的地址空间实现线性的，因此用物理上的相邻实现数据元素之间的逻辑相邻是非常合理的。顺序表结构如下图：

一维数组在内存中也是连续的存储空间，因此可以用一维数组来表示顺序表。
可以用一个java接口(interface)List.java来表示顺序表：
