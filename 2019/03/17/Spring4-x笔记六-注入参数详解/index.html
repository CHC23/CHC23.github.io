[Spring4.x笔记六] 注入参数详解

发表于 2019-03-17 | 分类于 Java Web

在Spring配置文件中，可以注入的参数有以下几种类型：

  • 字面值注入（基本数据类型）
  • 引用其他的Bean
  • 内部Bean注入
  • null值注入
  • 级联属性注入
  • 集合类型属性注入
1.字面值注入

字面值：一般值可用字符串表示的值，通过标签进行注入，在默认情况下基本数据类型及其封装类、String类型等都可以采取字面值注入的方式。

字面值注入

①中的<![CDATA[]]>:XML特殊标签，作用是让XML解析器将标签中的字符串当做普通文本来处理，防止特殊字符串破坏XML格式。
